# Contributing to Life Dashboard

Thank you for your interest in contributing! Here's how to get started.

## 🎯 Development Setup

```bash
git clone https://github.com/dhanush017/life-dashboard.git
cd life-dashboard
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## 📋 Before You Start

1. Check existing [Issues](https://github.com/dhanush017/life-dashboard/issues)
2. Fork the repository
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Test locally: `python main.py`
6. Commit: `git commit -m "Add feature description"`
7. Push: `git push origin feature/your-feature`
8. Open a Pull Request

## ✅ Code Standards

- Follow PEP 8
- Add docstrings to functions
- Write type hints where possible
- Test your changes

## 🐛 Found a Bug?

1. Check if it's already reported
2. Create a new Issue with:
   - Bug description
   - Steps to reproduce
   - Expected behavior
   - Your environment (OS, Python version)

## 💡 Feature Requests

1. Check existing discussions
2. Open a new Discussion explaining your idea
3. We'll discuss feasibility and design

## 📝 Commit Messages

- Use clear, descriptive messages
- Start with a verb: "Add", "Fix", "Update", "Remove"
- Example: "Add AI insights feature"

## 🎁 Thank You!

Every contribution helps make Life Dashboard better! 🚀
